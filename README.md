# STEAM / OVW

Steamで「圧倒的に好評」のゲームを抽出し、レビュー数が多い順に表示する非公式Webアプリです。

## 起動方法

1. Node.js 18以上をインストール
2. このフォルダで `npm start` を実行
3. ブラウザで `http://localhost:4173` を開く

外部ライブラリは使っていないため、`npm install` は不要です。

## 主な機能

- Steamの検索結果から「圧倒的に好評」を抽出
- レビュー数・高評価率・タイトルで並べ替え
- タイトル検索と最低レビュー数フィルター
- 25〜200件の取得件数切り替え
- 表示結果のCSV保存
- Steamに接続できない場合の内蔵サンプル表示
- スマートフォン対応の白黒・太字ゴシックUI

## 注意

Steamの検索ページ仕様が変わった場合は、`server.mjs` の解析処理を調整する必要があります。価格・評価・レビュー数は取得時点の情報です。このアプリはValve CorporationおよびSteamの公式アプリではありません。
